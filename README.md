# CrystalStudio
CrystalStudio - An easy game engine written in python. More features will be added soon! To request features, just open an issue.

## Wiki
Coming soon - We're working on it.

## Features
 - Inifinite scenes
 - Remove scenes
 - Add scenes
 - Edit labels (Text)
 - Edit buttons (Action, text and notes)
 - Remove buttons
